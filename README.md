GoChat -- приложение для общения в реальном времени

Запуск приложения:

```docker-compose up```

При желании можно настроить параметры запуска через .env  в корне проекта

---

Приложение находиться по адресу
http://localhost/

К сожалению, регистрация через сайт еще не настроена, предлагается добавлять юзеров через 

http://localhost:8080/swagger/index.html#/

или

через Postman (спека ./backend/docs/swagger.json)

По ручке POST /user/
